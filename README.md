# 人有三急 — Web 小游戏

一个像素风的轻松幽默小游戏。进入游戏后先选择你的角色形象，然后面对一排随机生成的男厕小便池，其中部分已被占位、部分空着。点击空位完成"选择"，获得轻松幽默的反馈。

## 玩法

1. 从 6 张角色卡片中选择你的形象（含随机抽取），进入厕所场景
2. 观察当前场景，在一排尿兜中挑选一个空位
3. 点击空位 → 玩家角色从左走入，到达后弹出随机夸赞（"完美！""明智之选！""讲究！"等），播放成功音效
4. 点击有人位置 → 玩家走入，所有占位 NPC 弹出嫌弃气泡：被点的 NPC 说"有病吧"，其他 NPC 说"变态"，播放拒绝音效
5. 点击非尿兜区域 → 玩家走入点击位置，所有 NPC 说"变态"
6. 成功后点击"下一泡"（或"我是变态"）刷新场景；不想选可点击"忍一时风平浪静"跳过
7. 连续忍 4 次触发彩蛋爆炸动画

## 角色预设

游戏提供 5 种固定角色 + 1 个随机选项：

| 角色 | 特征 |
|------|------|
| 黑皮体育生 | 深肤色、白色运动服、高个子 |
| 圆脸络腮胡 | 络腮胡、偏壮体型 |
| 精神帅小伙 | 金色配色、瘦高型 |
| 资深程序员 | 秃头、格子衫 |
| 清纯男大 | 蓝白配色、正常体型 |
| 随机 | 从上述 5 种中随机抽取 |

## 随机项

每次进入场景，以下要素随机组合：

| 随机项 | 可能性 |
|--------|--------|
| **尿兜数量** | 3 ~ 8 个 |
| **场景亮度** | 正常 / 半亮(傍晚) / 昏暗(夜晚) |
| **隔板** | 无 / 短隔板 / 长隔板 |
| **标语** | 无 / 每个尿兜上方小标语 / 墙中央大标语 |
| **儿童尿兜** | 无 / 最左侧 / 最右侧 |
| **占位 NPC** | 0 ~ (N-2) 人随机占位，确保至少 2 个空位 |
| **NPC 外观** | 10 套帽色+上衣色+发色配色 × 8 种裤子 × 4 种鞋子 |
| **NPC 体型** | 瘦 / 正常 / 壮，搭配随机身高 |
| **儿童 NPC** | 儿童尿兜前固定为矮小瘦型小孩，配色鲜艳 |
