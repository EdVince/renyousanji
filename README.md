# 人有三急 — Web 小游戏

一个像素风的轻松幽默小游戏。每次进入，面对一个随机生成的小便池队列，玩家只需点击一个空位，完成"选择"，获得轻松幽默的反馈。

## 源码目录介绍

```
├── audio                                     // 音频资源
├── images                                    // 图片资源（原项目遗留，实际使用 Canvas 程序化绘制）
├── js
│   ├── runtime
│   │   └── music.js                          // 音效管理器
│   ├── databus.js                            // 游戏状态管理
│   ├── main.js                               // 游戏主逻辑（场景生成、渲染、交互、动画）
│   └── render.js                             // Canvas 初始化及屏幕尺寸
├── game.js                                   // 游戏入口
```
